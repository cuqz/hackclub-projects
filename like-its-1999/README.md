# Like It's 1999

A website I built inside a Windows 98 emulator. No modern tools, just EditPlus, Paint Shop Pro, and Netscape the way people made sites back then.

## The site

An Attack on Titan fan page called TITANNET. Guestbook, visitor counter, marquee text, animated GIFs. The whole 90s package.

## How it works

1. Go to 99.hackclub.com
2. Click BOOT WINDOWS 98
3. Write the HTML in EditPlus
4. Make graphics in Paint Shop Pro
5. Test in Netscape
6. Submit before July 13 at 5PM Eastern

They mail you a real floppy disk with your code on it.
