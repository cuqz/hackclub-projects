# Kintsugi

Multi-agent community problem solver. Submit an issue, watch five AI agents analyze it, get a step-by-step solution.

## Quick start

Backend: cd backend && go run . (no API key needed — runs in mock mode)
Frontend: cd dashboard && npm install && npm run dev

## Stack
Go + SQLite + Featherless.ai / React 19 + Vite + shadcn/ui + Tailwind v4
