# Founder Cyberdeck

Handheld terminal thing inspired by Attack on Titan. Screen, mechanical keyboard, rotary encoders, joystick, Pi Pico inside. Case is 3D printed.

## Specs

180x100x30mm case. 3.5" TFT screen angled at 30 degrees. 15 keys in a 5x3 grid with Cherry MX switches. Two rotary encoders for scrolling, analog joystick for navigation. Pi Pico W running the show. Two swappable 18650s with a BMS. USB-C for power.
