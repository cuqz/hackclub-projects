# OneKey Runner

Geometry Dash-style runner that uses one key. Jump, dodge obstacles, don't die, get a high score.

## How to play

Open index.html in any browser. Press any key to jump. Avoid the spikes. Speed keeps going up. That's the whole game.

Same key restarts when you die.
